This is my Movie App .Here users can find information about movies.

The table below describes various endpoints along with their response and requests.

Buisness Logic URL HTTP method Request body data format Response body data format
List all movies. /movies GET none JSON object holding data about all movies.Example:

{{Title:"Titanic", Description:"The movie is about the 1912 sinking of the RMS Titanic. It stars Kate Winslet and Leonardo DiCaprio. The two play characters who are of different social classes. They fall in love after meeting aboard the ship, but it was not good for a rich girl to fall in love with a poor boy in 1912."},Director:{Name:"James Cameron",Bio:"Canadian",Birth:"Aug 16,1954"}, Genre:{"Name:Romance", Description:"Tale of romance between two people"}}

Get data about single movie by title. /movies/:Title GET none JSON object holding data about single movie by name.
Get genre of a movie by its title. /movies/genre/:Title GET none json object holding the movie's genre.
Get data about director by name. /movies/directors/:Name GET none json object holding director's data.Structure:

{Name:"Steven Spielberg",Bio:"American",Birth:"Dec18,1946"}
Add new user. /users POST JSON object holding data about new user. Structure:

{ Name: "Charles Ben", Password: "ABC123", Email: "charles.ben@gmail.com", Birthday: "05/29/87"}

JSON object holding data about new user. Structure:

{ id : 10,Name: "Charles Ben", Password: "ABC123", Email: "charles.ben@gmail.com", Birthday: "05/29/87"}

Update a user's information /users/:Name PUT JSON object holding data about the user to add. Structure:

{ Name: "Charles Ben", Password: "ABC123", Email: "charles.ben@gmail.com", Birthday: "05/29/87"}

A message indicating the user's information was updated.
Add a movie to the user's favorites list /favoritemovies/:Name/Movies/:MovieID POST None A message indicating the movie title was added to the user's favorites list.
Delete a movie from the user's favorites list /favoritemovies/:Name/Movies/:MovieID DELETE None A message indicating the movie title was deleted from the user's favorites list.
Remove a user from the registry. /users/:Name DELETE None A message indicating the user was removed from the registry.
