This is my Movie App .Here users can find information about movies.

The table below describes various endpoints along with their response and requests.

Buisness Logic URL HTTP method Request body data format Resonse body data format
List all movies. /movies GET none JSON object holding data about all movies.Example:

{name:"titanic",director:{name:"James Cameron",bio:"Canadian",dob:"Aug 16,1954"} genre:"Romance"}

Get data about single movie by title. /movies/:title GET none JSON object holding data about single movie by name.
Get data about a movie genre. /movies/:title/genre GET none json object holding the movie's genre.
Get data about director by name. /movies/:title/director/name GET none json object holding director's data.Structure:

{Name:"Steven Spielberg",Bio:"American",dob:"Dec18,1946"}
Add new user. /users POST JSON object holding data about new user. Structure:

{ username: "Charles Ben", password: "ABC123", email: "charles.ben@gmail.com", dateofbirth: "05/29/87"}

JSON object holding data about new user. Structure:

{ id : 10,username: "Charles Ben", password: "ABC123", email: "charles.ben@gmail.com", dateofbirth: "05/29/87"}

Update a user's information /users/:username PUT JSON object holding data about the user to add. Structure:

{ username: "Charles Ben", password: "ABC123", email: "charles.ben@gmail.com", dateofbirth: "05/29/87"}

A message indicating the user's information was updated.
Add a movie to the user's favorites list /favorites/:username/movies/:title POST None A message indicating the movie title was added to the user's favorites list.
Delete a movie from the user's favorites list /favorites/:username/movies/:title DELETE None A message indicating the movie title was deleted from the user's favorites list.
Remove a user from the registry. /users/:username DELETE None A message indicating the user was removed from the registry.
